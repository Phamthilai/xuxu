== Steps to Setup Theme ==

Below are the following step to setup theme static page.
=========================================================
Step 1. Add new page named as "home page" and select the template "Static Front Page".
Step 2. Go to customizer >> static front page >> check Static page, then select the page which you have added example "home page".

For SLider
==============
Step 1. Add new page, add title, content and featured image and then publish it.
Step 2. Go to customizer >> Theme Settings >> Slider settings >> here you can select the page which you have add for slider.

For Social Link
====================
Step 1. Go to customizer >> Theme Settings >> Social Icon Section >> here you can add your socail links.

For Topbar
==============
Step 1. Go to customizer >> Theme Settings >> Topbar Section >> here you can add your Contact details.

For Accomodation Section 
=========================
Step 1. Add new page, add title, content and featured image and publish it.
Step 2. Go to customizer >> Theme Settings >> Accomodation Section >> here you will get option to add title, sub title , and select pages which you have added.

For General Settings
======================
Step 1. Go to customizer >> Theme Settings >> General Settings >> here you can change the layout of the blog post.
